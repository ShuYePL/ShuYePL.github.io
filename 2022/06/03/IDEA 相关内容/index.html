下载安装 IDEA

进入 IDEA 官网，下载 IDEA 的安装包

IDEA 有两个版本，一个是专业版（要注册账号购买，或者注册学生账号），一个是社区版（免费），两个版本的安装过程都是差不多的，我这里就选择专业版了，点击下载按钮，等一小会浏览器就开始下载了

双击下载得到的安装文件

点击 Next >

点击 Browse… 选择安装的目录，然后点击 Next >

  1. 选择是否添加桌面快捷方式（推荐）
  2. 选择是否添加环境变量（推荐）
  3. 选择和 IDEA 关联的文件类型（看个人喜好）

点击 Next >

等待安装完毕

点击 Finish 完成安装

接下来的就是打开 IDEA ，登录账号，登陆账号要在网页中进行，登录完成之后的页面是这样的

注册 IDEA 学生账号

IDEA 专业版的免费试用时间只有 30 天，在查询免费延长时间的过程中，发现可以申请学生账号，免费试用很长的时间。

申请网址

下拉页面，在页面下方找到一个很显眼的蓝色 Apply now 的按钮，点击它

点击后跳转到一个界面，这个界面我跳转过来的时候是中文的，要是你跳转过来是英文的话，可以在网页上面导航栏处修改语言

接着，下面的东西就全是中文了，所以按照中文的提示填写相关内容就好了，注意这里像我一样是学生的话是需要一个学校的教育邮箱的，有些学校的邮箱验证可能不通过，还需要官方文件进行认证，这个时候按照要求填写信息，提交学生证等资料，然后其中有个学信网报告在线验证码的，是登录学信网之后点击在线验证，选择相应的验证方式，然后申请就可以得到了，之后点击下方的提交申请按钮提交申请即可，它提交之后要真人认证，所以需要一点时间，慢慢等待即可。

时间：2022年5月31日23:10:35

我终于收到邮件了，太激动了，都已经过去十几二十天了，前面好像是我邮箱写错了，所以前几天重新申请了一份，这次过了两三天就收到了邮件，哈哈哈~

点击上图红色方框中的链接，跳转到一个协议之类的界面，慢慢往下拉，拉到底部的时候底部旁边的蓝色 I Accept 按钮就会由暗变亮，点击它就可以跳转到一个注册或者登录的界面，因为我还没有账号，所以，我选择注册一个账号，也就是在下面的方框中输入邮箱，然后点击 Sign up ，如果你有账号的话，那就直接在上面填入账号和密码，然后点击 Sign in

然后登录你注册账号的邮箱，在收件箱中查看最新收到的一封邮件，如下，点击图中红色方框内链接 Comfirm your accout

跳转到一个页面，在这个页面中填写相关信息，就是名字之类的，然后，在下面中的 Password 中写入你想设置的账号密码，Repeat Password 的意思就是再写次输入你的密码

把下图中的 √ 给点上，然后点击 Submit

现在，去 IDEA 登录这个账号就行了，账号就是你注册 IDEA 账号用的邮箱。

调节字体的大小

点击 File -> Settings… 打开如下所示的界面

选择 Editor -> Font ，在 Size 处修改数字的大小即可调整字体大小，在右边可以预览字体的效果，确认好要调整的效果，点击右下角的 Ok 按钮就完成了。

IDEA创建空工程时自动携带同名模块解决方案

IDEA 版本是 2022.1.1 ，在创建空工程的时候，老是会在工程里面自动携带一个和工程同名的模块。

解决方法

将同名模块的 .iml 文件直接删除就可以了，然后需要注意的是在创建新模块时，模块的本地地址要写好。

IDEA配置数据库可视化

点击编辑界面右侧的 Database –> + –> Data Source –> MySQL

在下面的界面中填写数据库的用户名称，密码，数据库名称，然后点击 Test Connection 测试数据库连接是否正常，注意，在第一次点击这个测试时可能会提示你没有相关的文件，这个时候选择下载相应文件的选项就可以了

点击 Ok ，即可在 IDEA 中查看可视化后的数据表

IDEA-Cannot run program DJava1.8… 系统找不到指定的文件

前两天把 D 盘清空，和 C 盘合并了，而之前的 Java 安装在 D 盘中，今天运行 Mava 逆向工程时出现了一点小错误。

打开 Project Structure –> SDKS 将之前失效的 SDK 删除，修改成新的 SDK 位置，然后出现了这个错误，可以从描述里面看到是 Maven 的 Runner 配置有问题

File –> Settings –> Build, Execution, Deployment –> Build Tools –> Maven –> Runner ， 点开进去 JRE 那里爆红了，修改一下即可。
